# Advent of Code 2020

Repository with solutions until day 4.

Install repository:
```sh
git clone https://github.com/bedomax/katas-advent-of-code.git
cd katas-advent-of-code
bundle install
```
Run the tests:
```sh
# Using rspec
rspec spec/modules/**/* --format NyanCatFormatter
```

Run main program:
```sh
ruby lib/main.rb
```
